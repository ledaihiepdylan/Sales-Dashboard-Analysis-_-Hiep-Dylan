# 🧩 Shopify Dashboard Feature Description

The dashboard is designed to monitor the entire user lifecycle of the application through four core KPIs:

Install
Uninstall
Paid Users
Churn (Paid User Drop-off)

Users can interact dynamically with the dashboard via global filters and four main analytical pages.

## ⚙️ 1. Global Filters & Navigation

The dashboard provides key control components:

🔘 Select Metric
Allows users to quickly switch between the four KPIs:
Install / Uninstall / Paid / Churn
👉 Enables focused analysis at each stage of the user funnel

📄 Select Page
Navigation across four main pages:

Overview
Map
Plus Plan
Detail

🌍 Country Filter
Filters data by geographic location (country-level segmentation)

📅 Time Filter
Supports filtering by:

Specific years (e.g., 2024, 2025)
Or full historical data

## Dashboard Page

### 📊 2. Overview Page

🎯 Objective:
Provide a high-level view of overall system performance

📌 Key Features:

Display of primary KPIs:
Total values
Average daily metrics
Month-over-month (MoM) growth rate
Mini trend charts for time-series visualization
Breakdowns / Dimensions:
Attribution (Organic vs Paid traffic)
Shop Plan (user segmentation by subscription tier)
Shop Domain (user type classification)
Shop Name (top-performing stores)

👉 This page is primarily used to:

Quickly monitor overall performance
Detect anomalies such as drops or spikes in key metrics

### 🌍 Page 2: Map – Geographic Analysis

🎯 Objective

The Map page enables geographic performance analysis of users at the region/country level, allowing stakeholders to:

Identify high-potential markets for growth
Detect regions with high churn rates
Compare user behavior across different countries

👉 This supports data-driven market expansion decisions and localized user strategy optimization
